# Serialize a TOML value

Serialize a TOML value

## Usage

``` r
ts_serialize_toml_value(obj, options = NULL)
```

## Arguments

- obj:

  R object to serialize as a TOML value.

- options:

  A named list of `tstoml` options, see
  [`tstoml_options()`](https://gaborcsardi.github.io/tstoml/reference/tstoml_options.md).
